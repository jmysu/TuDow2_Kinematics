# TuDow2_Kinematics 土豆2號-仿生動力學
My Robot TuDow2 simulation w/ Kinematics<br/>
羅拔特 是用QT5+OpenGL+QGLViewer 畫出來的！<br/>
茶壺位置 是用 __老歐的餘弦定理__ 算出來的！<br/>
如果位置算錯了 那可能要叫他起來 說明白講清楚！！！<br/><br/>
